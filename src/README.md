#Financials

> Financial algorithms implemented in TypeScript

##Setup

####Install

```sh
git checkout https://github.com/kcjmowright/financials.git
npm install
```

####Test

```sh
npm test
```

####Build Live

```sh
npm run start
```

####Clean

```sh
npm run clean
```
